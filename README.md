University Learning Management System

Overview
A full-scale Agile-based Learning Management System inspired by Canvas.
This system simulates role-based academic workflows including course management and user interaction.

Features
- Role-based access (Admin, Instructor, Student)
- Course creation and management
- Assignment tracking
- UI/UX prototype built in Figma
- Agile sprint planning (Jira, Trello)

Tech Stack
- Languages: Java / HTML-CSS
- Concepts: OOP, SDLC, Agile Methodology
- Tools: IntelliJ, Figma, Jira, Trello

What I Learned
- Implementing Agile workflow in a team environment
- Integrating frontend and backend modules
- Translating user requirements into system design
